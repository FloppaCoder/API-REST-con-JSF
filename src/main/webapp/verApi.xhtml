Como Funciona Nuestra API REST?

Nuestro Servicio Web Rest proporciona respuestas a solicitudes POST y GET, esto mediante URL's con instrucciones especficas, pudiendo asi acceder a 5 metodos principales.

GETObtener Todos los Insumos

URL

/api/insumoMed

Respuesta

{

"cantidad": 40,

"id": 1,

"nombre": "Mascarillas quirúrgicas",

"precio": 3.5

},

{

"cantidad": 22,

"id": 2,

"nombre": "Alcohol gel",

"precio": 2.45

} ... y demás registros...

GETObtener Insumo por su ID

URL

/api/insumoMed/{id}

Respuesta

{

"cantidad": 40,

"id": 1,

"nombre": "Mascarillas quirúrgicas",

"precio": 3.5

}

POSTCrear un Nuevo Insumo

URL

/api/insumoMed/create

Datos de Entrada

{

"cantidad": 40 (Numero entero),

"nombre": "Acetaminofen" (cadena de texto),

"precio": 3.5 (Numero entero o Decimal de dos digitos)

}

POSTModificar un Insumo Existente

URL

/api/insumoMed/update/{id}

Datos de Entrada

{

"cantidad": 40 (Numero entero),

"nombre": "Acetaminofen" (cadena de texto),

"precio": 3.5 (Numero entero o Decimal de dos digitos)

}

DELETEEliminar un Insumo por ID

URL

/api/insumoMed/delete/{id}

Respuesta

Insumo Eliminado Correctamente
